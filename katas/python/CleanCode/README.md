# Clean Code - Python

Ejemplos de principios de Clean Code aplicados en Python.

## Contenido

1. **naming/** - Nombres descriptivos y sin magic numbers
2. **functions/** - Funciones con una sola responsabilidad
3. **format/** - Formato consistente y legible

- `README.md` - Guía del concepto
- `*_bad.py` - Ejemplo que viola el principio
- `*_exercise.py` - Tu práctica
