# OOP - Python

Ejemplos de los 4 pilares de la Programación Orientada a Objetos aplicados en Python.

## Contenido

1. **abstraction/** - Ocultar detalles de implementación
2. **encapsulation/** - Proteger el estado interno
3. **inheritance/** - Reutilización de código
4. **polymorphism/** - Comportamiento específico sin condicionales

Cada carpeta contiene:

- `README.md` - Guía del concepto
- `*_bad.py` - Ejemplo que viola el principio
- `*_exercise.py` - Tu práctica
- `*_good.py` - Ejemplo que sigue el principio
