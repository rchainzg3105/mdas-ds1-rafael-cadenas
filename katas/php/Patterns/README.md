# Patrones de Diseño en PHP

Soluciones probadas para problemas comunes de diseño de software.

## Contenido

- **factory/** - Patrón creacional para creación de objetos
- **builder/** - Patrón creacional para construcción compleja
- **adapter/** - Patrón estructural para compatibilidad de interfaces
- **strategy/** - Patrón de comportamiento para selección de algoritmos

Cada carpeta contiene ejemplos malos, buenos y ejercicios para practicar.
